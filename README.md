

# 🧠 Basic Neural Network from Scratch (NumPy)

## 📌 Project Overview

This project implements a **basic neural network from scratch using only NumPy**, without relying on any machine learning libraries such as TensorFlow or PyTorch. The model is trained to classify handwritten digits from the MNIST dataset.

This project was developed as part of my work as an **AI Intern at CoreGen**.

---

## 🏢 Organization

**CoreGen**

---

## 🎯 Objective

* Build a neural network from scratch
* Understand forward and backward propagation
* Implement activation functions manually
* Train the model and evaluate performance

---

## 🛠️ Tech Stack

* Python
* NumPy
* Matplotlib
* Scikit-learn (for dataset loading only)

---

## 🧩 Features

* Fully connected neural network (2 layers)
* ReLU activation function
* Softmax output layer
* Cross-entropy loss
* Gradient descent optimization
* Accuracy evaluation
* Loss visualization

---

## ⚙️ Model Architecture

* Input Layer: 784 neurons (28×28 images)
* Hidden Layer: 64 neurons (ReLU)
* Output Layer: 10 neurons (Softmax)

---

## 🔄 Workflow

1. Load and preprocess MNIST dataset
2. Normalize input data
3. Apply one-hot encoding to labels
4. Initialize weights and biases
5. Perform forward propagation
6. Compute loss (cross-entropy)
7. Perform backpropagation
8. Update parameters using gradient descent
9. Evaluate accuracy on test data
10. Visualize training loss
